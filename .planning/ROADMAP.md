# Roadmap: School Management System (SMS)

## Milestone 1: Core Academic Workflows

### Phase 1: Attendance System
Implement the daily attendance marking and reporting system.
- **Goal**: Enable teachers to track student presence efficiently.
- **Plans**:
    - [ ] Create attendance data structures in Motoko.
    - [ ] Build teacher attendance marking UI.
    - [ ] Implement student/admin attendance reporting views.

### Phase 2: Exams & Results
Implement exam scheduling, marks entry, and automated grading.
- **Goal**: Digitize the academic assessment process.
- **Plans**:
    - [ ] Enhance Motoko backend for exam management.
    - [ ] Build exam scheduling interface for Admins.
    - [ ] Create marks entry form for Teachers.
    - [ ] Implement result calculation and student result view.

### Phase 3: Dashboard Refinement & Polishing
Enhance the user experience across all role-based dashboards.
- **Goal**: Ensure the system is intuitive and responsive.
- **Plans**:
    - [ ] Refactor existing dashboards for better data visualization.
    - [ ] Improve mobile responsiveness.
    - [ ] Add final UI polish and error handling.
