# Stack

## Core
- **Frontend**: React 18.2.0, Vite 5.1.6
- **Language**: TypeScript 5.2.2
- **Backend (ICP)**: Motoko (Dfinity SDK / DFX)
- **Backend (External)**: Supabase (supabase-js 2.89.0)

## Styling & UI
- **CSS**: Tailwind CSS 3.4.17, PostCSS 8.5.6
- **Components**: Lucide React (Icons), Radix UI (implied by utility classes), Framer Motion (Animations)
- **Utility**: clsx, tailwind-merge, class-variance-authority

## State & Data
- **Data Fetching**: TanStack React Query 5.90.16
- **Routing**: React Router DOM 6.22.0
- **UI Components**: next-themes, sonner (Toasts)

## Specialized
- **Images**: browser-image-compression
- **QR**: qrcode.react
- **Media**: react-webcam
- **Printing**: react-to-print
